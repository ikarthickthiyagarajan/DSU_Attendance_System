# DSU Attendance Management System

A modern, user-friendly attendance management solution designed specifically for Dhanalakshmi Srinivasan University. This application streamlines the process of tracking and managing student attendance across departments and courses.

## Features

- **Secure Authentication**: Role-based access for administrators, faculty, and students
- **Real-time Tracking**: Instantly record and update attendance data
- **Comprehensive Dashboard**: Visual analytics and attendance patterns
- **Mobile Responsive**: Accessible across devices for on-the-go management
- **Automated Reporting**: Generate attendance reports by class, student, or date range
- **Notification System**: Alerts for low attendance and important updates

Built with React 19, Next.js 15, and Tailwind CSS, this application offers a sleek, intuitive interface with powerful functionality. The system helps improve academic accountability while reducing administrative overhead through intelligent automation.
